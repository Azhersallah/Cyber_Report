# Photo Printer Pro

بەرنامەیەکی پیشەیی بۆ چاپکردنی وێنە، تایبەت بە کارگەکانی چاپ.

## تایبەتمەندییەکان

- چاپکردنی وێنە بە قەبارە جیاوازەکان
- پشتگیری لە لەیاوتی جیاواز
- دەستکاریکردنی وێنە
- چاپکردنی کارتی بزنس
- چاپکردنی پسوولە
- چاپکردنی وێنەی ناسنامە
- **دروستکردنی QR Code** - دروستکردن و چاپکردنی QR Code بۆ:
  - تێکست
  - URL/وێبسایت
  - ئیمەیڵ
  - ژمارەی تەلەفۆن
  - SMS
  - زانیاری WiFi
  - شوێنی جوگرافی (GPS)
  - گۆڕینی ڕەنگ و قەبارە
  - زیادکردنی لۆگۆ لە ناوەڕاست
  - دابەزاندنی وەک PNG
  - چاپکردن
